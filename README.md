# UPTD SDN 1 GABUS KULON - DIGITAL 2026

Website resmi dan pusat arsip digital UPTD SDN 1 GABUS KULON. Media informasi, dokumentasi kegiatan, dan prestasi sekolah secara modern dan profesional.

## Fitur Utama

- **Arsip Digital:** Manajemen dokumen dan file sekolah yang terorganisir.
- **Multibahasa:** Mendukung Bahasa Indonesia dan English.
- **Dashboard Admin:** Kelola konten, kategori, dan log sistem.
- **Berita & Galeri:** Informasi terkini dan visual kegiatan sekolah.

## Pengembangan Lokal

**Prasyarat:** Node.js (v18+)

1. Clone repositori ini:
   ```bash
   git clone <repository-url>
   ```
2. Instal dependensi:
   ```bash
   npm install
   ```
3. Konfigurasi Environment:
   Buat file `.env` dan tambahkan variabel yang diperlukan (lihat `.env.example`).
4. Jalankan aplikasi:
   ```bash
   npm run dev
   ```

## Integrasi & Deployment

### GitHub
- Hubungkan folder proyek ini ke repositori GitHub baru.
- Pastikan `.gitignore` sudah mencakup `node_modules` dan `dist`.

### Vercel
Aplikasi ini sudah dikonfigurasi untuk deployment ke [Vercel](https://vercel.com).
1. Hubungkan akun Vercel Anda ke repositori GitHub.
2. Tambahkan variabel lingkungan (Environment Variables) di dashboard Vercel jika diperlukan (seperti Firebase Config).
3. Vercel akan otomatis mendeteksi konfigurasi Vite dan membangun aplikasi dari folder `dist`.
